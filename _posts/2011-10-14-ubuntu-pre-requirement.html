Ubuntu just released a major update, the newest stable version of Ubuntu is 11.10 with code name Oneiric Ocelot, this version brings a lot of major improvement to the unity (interface) and its desktop shell (as opposed to Gnome Shell) I've been mentioned previously. 


Before Installing Ubuntu 11.10, couple of things worth mentioning. 

  • Ubuntu 11.10 ships with Unity, but by default is really Unity 3D. Due to the high rendering of Unity, lower end hardware aren't supported in Unity's 3D. Ubuntu however was clever to automatically enable unity 2D that is supported by most hardware at the log on screen, or if you prefer the default Gnome Shell, you can simply install it with the following command 

sudo apt-get install gnome-shell

  • All as shown below. 
  • Here is Ubuntu's official database of compatible hardware for Unity3D
  • You can also invoke a script pre-installed in Ubuntu 11.10 to check if your hardware can run Unity with compiz by running the following command 

/usr/lib/nux/unity_support_test -p
